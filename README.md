# VR Workstation

Company: 5xLiving
Project: VR Workstation
Status: Initial project shell

## Purpose
A stack-neutral starter workspace for building VR workstation features and related tooling.

## Structure
- `assets/` shared assets and media
- `config/` environment and project configuration
- `docs/` project notes and requirements
- `src/` application source code

## Next Build Paths
- Add a web client under `src/` if this becomes a browser-based control surface.
- Add Unity-specific folders if this becomes a Unity-driven VR project.
- Add backend services only when the API contract is defined.
